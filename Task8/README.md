### Sales Performance Dashboard — Power BI



#### Overview:



This project presents an interactive Power BI dashboard built using the Superstore Sales dataset.

The dashboard provides a clear view of sales, profit, and category performance across different regions and time periods, helping decision-makers identify trends and optimize business strategies.



#### Objective:



To analyze sales data and visualize:



Monthly sales performance



Regional sales distribution



Category-wise contribution



Profitability trends and KPIs



#### Tools Used:



Power BI Desktop — for visualization and dashboard design



Microsoft Excel / CSV — dataset source



DAX (Data Analysis Expressions) — for calculated metrics



(Optional) Python (Pandas) — for data preprocessing



#### Dataset Information:



File: Superstore\_Sales.csv

Key Columns:



Column		Description

Order Date	Date when the sale occurred

Region		Geographical region of sale

Category	Product category

Sales		Total sales amount

Profit		Profit earned from sales



#### Additional Calculated Columns:



Month-Year — formatted as “MMM YYYY” for trend analysis



Profit Margin (%) — (Profit / Sales) \* 100



Visuals Created



KPI Cards: Total Sales, Total Profit, Profit Margin



Line Chart: Monthly Sales Trend



Bar Chart: Sales by Region



Donut Chart: Sales by Category



Slicer: Filter by Region and Year



#### Key Insights:



West region recorded the highest sales performance.



Technology category contributed the most profit overall.



Sales peaked during Q4 (Nov–Dec) due to seasonal demand.



Profit margins were lower in the South region, suggesting cost inefficiency.



Strong positive correlation between sales and profit, but optimization needed in certain areas.



#### Dashboard Design Highlights:



Clean layout with light background and consistent color palette



Interactive slicers for region/category filters



Dynamic titles and tooltips for user-friendly navigation



KPI cards provide quick performance overview
